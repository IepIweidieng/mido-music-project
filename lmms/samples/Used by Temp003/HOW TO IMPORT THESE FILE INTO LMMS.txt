第一步：根據說明文件指示，下載或安裝以下兩個檔案。
Step 1: Download or install the following two files with the instructions in the instruction files.

OTT_x64
TOUHOU INSTRUMENT + DRUM KIT.sf2

第二步：移動此資料夾到 LMMS 工作目錄下的「samples」資料夾中。
Step 2: Move this folder into "samples" folder under LMMS's work directory.

第三步：將下載完畢的 sf2 檔案移動到 LMMS 工作目錄下的「samples/soundfonts」資料夾中。
Step 3: After finishing the downloading, Move the sf2 files into "samples/soundfonts" folder under LMMS's work directory.

第四步：使用 LMMS 1.2.0-rc3 或更新的版本開啟專案。
Step 4: Open the project in LMMS with version 1.2.0-rc3 or newer.
